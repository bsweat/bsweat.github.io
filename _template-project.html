Project Card Template

Copy the object below, fill in your details, and paste it into the projects array in projects.html. Cards auto-number — just add to the bottom and the count updates itself.

How to add a project

step_01Open projects.html in any text editor.
step_02Find the line that says // ── PROJECT DATA — add new projects to this array in the <script> block.
step_03Scroll to the end of the last object in the array (look for the last }, before the closing ];).
step_04Paste the filled-in template object after that last },
step_05Save. The new card appears automatically at the bottom of the grid, numbered correctly.

Template — copy and fill in

// ─── PASTE THIS AFTER THE LAST }, IN THE projects ARRAY ───────────────────
{
  status: 'wip',        // 'active' | 'wip' | 'done'  — see badge reference below

  name: 'Your Project Name',

  desc: 'One or two sentences describing what the project does, the problem it
        solves, and anything interesting about how it works.',

  stack: [
    { label: 'Python',  hi: true  },  // hi: true  = green highlighted tag (use for primary tech)
    { label: 'Nmap',    hi: false },  // hi: false = muted gray tag (use for secondary tech)
    { label: 'Shodan',  hi: false },
  ],

  link: {
    text: 'github →',    // 'github →' | 'view →' | 'demo →' | 'writeup →'
    href: 'https://github.com/yourrepo',  // use '#' if not live yet
  },
},
// ─── END PASTE ─────────────────────────────────────────────────────────────

Status badge reference

Active status: 'active'    In Progress status: 'wip'    Done status: 'done'

Stack tag reference

Primary Tech hi: true    Secondary hi: false
// Tip: Use one hi: true tag for the main language or tool, and hi: false for everything else. Keep the stack to 3–4 tags so the card stays readable.
