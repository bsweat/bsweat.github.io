Blog Post Template

Copy the object below, fill in your details, and paste it into the posts array in blog.html. Add it at the very top of the array so it appears first in the list.

How to add a post

step_01Open blog.html in any text editor.
step_02Find the line const posts = [ inside the <script> block.
step_03Paste your filled-in object immediately after the opening [ — before the first existing post. This puts it at the top of the list.
step_04Set id to the highest existing id number plus 1. Check the current posts — as of writing the highest is 7, so your next one is 8.
step_05Write the body content using the HTML elements in the reference section below. Save and open blog.html to verify.

Template — copy and fill in

// ─── PASTE THIS AT THE TOP OF THE posts ARRAY, RIGHT AFTER:  const posts = [ ────
{
  id: 8,                     // increment from the highest id in the array (currently 7)
  date: 'Apr 2026',         // format: 'Mon YYYY'
  title: 'Your Post Title Here',

  excerpt: 'One or two sentences shown in the post list. Keep it punchy — under 160 characters.',

  tags: 'security',           // space-separated: 'security' | 'ctf' | 'tools' | 'general'
                              // example with multiple: 'ctf security'

  tagHTML: '<span class="tag security">Security</span>',
                              // see tag HTML reference below

  kicker: '// Category · Subcategory',
                              // small label above the title, e.g. '// TryHackMe · CTF Writeup'

  body: \`
    <h2>Section Heading</h2>
    <p>Paragraph text. Wrap key terms in <em>emphasis tags</em> — they render bold and bright.</p>

    <h2>Code or Commands</h2>
    <pre>nmap -sV -sC --open <TARGET_IP></pre>

    <h2>Bullet List</h2>
    <ul>
      <li>First point</li>
      <li>Second point with <em>emphasis</em></li>
    </ul>

    <div class="note"><strong>// Note:</strong> Callout or takeaway text goes here. Great for warnings, tips, or TL;DRs.</div>
  \`,
},
// ─── END PASTE ─────────────────────────────────────────────────────────────────

Tag HTML reference — copy the snippets you need

Filter tags (tags field)

'security'General security topics, writeups, tools

'tools'Tool guides, lab setup, how-tos
'general'Everything else

tagHTML snippets

<span class="tag security">Security</span> green border
<span class="tag ctf">CTF</span> red border
<span class="tag">HomeLab</span> muted gray — any label
<span class="tag">OSINT</span> chain as many as you like
Security CTF HomeLab OSINT Nmap ← live previews of each tag style

Body HTML elements reference

Elements

<h2>Heading</h2> Green uppercase section heading
<h3>Sub-heading</h3> Smaller bold heading
<p>Text</p> Body paragraph, muted color
<em>word</em> Bold bright highlight inside a <p>
<pre>command</pre> Dark code block with green left border
<ul><li>...</li></ul> Bullet list with → arrows
<div class="note">...</div> Dashed callout box
<a href="...">link</a> Green link, use target="_blank" for external
